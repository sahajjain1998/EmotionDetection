# Sahaj
this project determines the emotions of a person based on their speech input

Software Used:
1.MATLAB 2014
2.PRAAT

Database:
1.we hace used berlin database which is available on the internet for free.
2.we can also create out own database using PRAAT software.

STEPS FOR CREATING PROJECT:
1.download matlab version 2014 for each function of this project to work
2.place each code in different matlab files under the same directory.
3.change the database URL according to the location where you have saved the database.
4.run the matlab project.
